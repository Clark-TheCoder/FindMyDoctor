# FindMyDoctor

FindMyDoctor is a simple React application built to demonstrate the use of **React** and **Bootstrap** for UI components.

## Features

- Uses **React** for component-based UI development.
- Styled with **Bootstrap** for responsive and consistent design.
- Loads doctor data from a local **JSON file**.
- Users can:
  - Filter doctors by **province**.
  - Select a doctor to view more details on a dedicated page.
- Demonstrates routing between pages using React Router.
